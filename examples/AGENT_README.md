# Examples

## Purpose

Reference integration layouts for major hosts and domain environments.

## Structure

Each subdirectory shows install, config, commands, a benchmark example, and troubleshooting.

## Where To Look

- host-focused examples: `claude/`, `codex/`, `copilot/`, `gemini/`
- embedded examples: `openhands/`, `sweagent/`, `aider/`, `langgraph/`
- domain examples: `shopify/`, `beseam/`
